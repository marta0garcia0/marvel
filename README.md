# Marvel challenge

The app is written in react + typescript and is launched with vite. For the tests we are using testing-library and jest.

## Installation

Installation

```bash
npm install
```

## Usage

To run the app in development mode url: http://localhost:5173/

```bash
npm run dev
```

To execute tests

```bash
npm run test
```
